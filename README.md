- HTML
- CSS
- Bootstrap
- Font Awesome


## Installation
1. Clone this repository :

	```
    $ git clone https://github.com/Mudanya/Corporate-Website.git
    ```

2. CD into the project folder on your machine

	```
    $ cd Corporate-Website
    ```
